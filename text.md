1º Bimestre – Introdução ao PHP
O PHP (Hypertext Preprocessor) é uma linguagem de programação voltada para o desenvolvimento web, utilizada principalmente no lado do servidor. Isso significa que o código PHP é processado no servidor antes de ser enviado ao navegador do usuário.
No 1º bimestre, entendemos a estrutura básica da linguagem, como a utilização de variáveis, tipos de dados (inteiros, strings, etc.), operadores e estruturas condicionais como if e else. Também foi importante aprender como inserir código PHP dentro de páginas HTML.
Outro ponto fundamental é compreender como o PHP interage com formulários, permitindo capturar dados inseridos pelo usuário. Esse conhecimento é essencial para a criação de sistemas dinâmicos, como cadastros e login.
 
1º Bimestre – Introdução ao Java
Java é uma linguagem de programação orientada a objetos, amplamente utilizada no desenvolvimento de sistemas, aplicativos e até jogos. Sua principal característica é a portabilidade, ou seja, um programa Java pode rodar em diferentes sistemas operacionais.
Durante o 1º bimestre, são abordados os conceitos iniciais, como a instalação do ambiente de desenvolvimento, a estrutura básica de um programa Java e o uso do método main. Também foram estudados tipos de dados, variáveis, operadores e estruturas de decisão (if, else,etc.).
Além disso, o aluno começa a entender o conceito de orientação a objetos, com noções básicas de classes e objetos, que são fundamentais para o desenvolvimento em Java.

1º Bimestre – Introdução ao C#
C# (C Sharp) é uma linguagem de programação desenvolvida pela Microsoft, muito utilizada na criação de aplicações desktop, web e jogos (especialmente com a Unity). É uma linguagem moderna e orientada a objetos.
No 1º bimestre, aprendemos os conceitos básicos da linguagem, como variáveis, tipos de dados, operadores e estruturas condicionais (if, else). Também são introduzidos os conceitos de entrada e saída de dados.
Além disso, são apresentados os fundamentos da programação orientada a objetos, como classes e métodos, preparando o aluno para desenvolver aplicações mais complexas nos próximos bimestres.

1º Bimestre – Aplicativos 1: Introdução ao HTML e CSS
Na disciplina de Aplicativos 1, nós tivemos o primeiro contato com o desenvolvimento de páginas web utilizando HTML e depois o CSS. Essas duas tecnologias são fundamentais para a criação de sites, sendo o HTML responsável pela estrutura e o CSS pela parte visual.
Durante o 1º bimestre, nos foram apresentados os conceitos básicos do HTML, incluindo a estrutura de uma página com as tags <html>, <head>, <body>, etc. nós “aprendemos” a utilizar elementos como títulos (<h1> a <h6>), parágrafos (<p>), listas (<ul> e <ol>), links (<a>) e imagens.
Em conjunto, foi introduzido o CSS, que permite estilizar os elementos da página. São trabalhadas propriedades como cores, fontes, tamanhos, espaçamentos e alinhamentos, tornando o conteúdo mais organizado e visualmente agradável.
Também são exploradas as formas de aplicar CSS:
Inline (direto na tag),
Interno (na seção <style>),
Externo (arquivo .css).
Ao final do bimestre, os alunos são capazes de desenvolver páginas simples, estruturadas e com uma apresentação visual básica.
