Web Foundations

O desenvolvimento de aplicativos começa pela web. O primeiro semestre foca na dupla dinâmica: HTML para estrutura e CSS para design.

Evolução Técnica

  • HTML5 Semântico: Aprendizado das tags estruturais que dão significado ao conteúdo para navegadores e SEO.
  • Elementos de Interface: Implementação de menus, listas, imagens responsivas e tabelas.
  • Design System e CSS3: Manipulação de tipografia, cores, espaçamentos e o modelo de caixa (Box Model).
  • Estratégias de Estilo: Entendimento de quando usar CSS Inline, Interno ou arquivos Externos para otimização.

O impacto visual

Uma boa estrutura é apenas metade do caminho. O design visual (CSS) é o que transforma bytes em experiências memoráveis para o usuário final.
