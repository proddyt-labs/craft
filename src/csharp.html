C# Ecosystem

O C# (C-Sharp) é a linguagem de elite da Microsoft. Elegante, moderna e extremamente versátil, ela é o coração do ecossistema .NET.

Currículo do 1º Bimestre

  • Arquitetura Microsoft: Introdução ao funcionamento do Common Language Runtime (CLR).
  • Entrada e Saída Dinâmica: Domínio dos comandos Console.ReadLine e Console.WriteLine para automação.
  • Controle de Fluxo: Implementação de algoritmos complexos usando estruturas de decisão aninhadas.
  • Exploração de Objetos: Primeiros contatos com Métodos e a importância da organização estrutural em POO.

Onde usar C#?

Se você quer criar jogos na Unity, aplicativos desktop para Windows de alta performance ou APIs escaláveis com ASP.NET Core, o C# é o seu caminho definitivo.
