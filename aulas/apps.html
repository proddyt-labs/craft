Fundamentos da Web

O desenvolvimento de aplicativos geralmente começa pela web, onde é possível validar ideias rapidamente, alcançar um público amplo e iterar com mais facilidade antes de investir em soluções mais complexas como apps nativos. O primeiro semestre foca na tripla dinâmica: HTML para estruturação, CSS para design e o JavaScript, que é uma linguagem de programação versatil e leve baseada em objetos, que serve para adicionar a interatividade.

Evolução Técnica

  • HTML5 Semântico: Uso de tags estruturais que atribuem significado ao conteúdo da página, facilitando a compreensão por navegadores, mecanismos de busca (SEO) e tecnologias assistivas(que leem o conteúdo em voz alta para pessoas com deficiência visual).
  • Elementos de Interface:Implementação de componentes como menus, listas, imagens responsivas e tabelas, visando organizar o conteúdo e proporcionar uma navegação clara e eficiente ao usuário.
  • CSS: Responsável pela estilização das páginas, permitindo a manipulação de tipografia, cores, espaçamentos e do modelo de caixa (Box Model), que define como os elementos são estruturados e ocupam espaço na interface.
  • Estratégias de Estilo:Compreensão das melhores práticas para aplicar CSS — seja inline, interno ou em arquivos externos — visando organização, reutilização de código e otimização do desempenho.

O impacto visual

E uma boa estrutura é apenas parte do processo. O design visual, por meio do CSS, é responsável por transformar código em experiências envolventes e memoráveis para o usuário final.
